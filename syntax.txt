Land is a fairly simple programming language, its syntax is somewhat inspired by the Python programming language. However, it has been made different from other programming languages, for example many programming languages have an obligation, whereas Land has no requirement.



Expressions and flow control

Sample statements from Land:



The IF statement executes a block of code together with else and elif (short for else-if) depending on a certain condition.
The FOR statement iterates through a iterable object, while assigning each item in the corresponding block to a local variable.
The WHILE statement executes that block of code as long as its condition is true.
The function is defined with the FUNC statement.
The PRINT statement acts as a screen print.

Turkish:

Land, oldukça basit bir programlama dilidir, sözdizimi biraz Python programlama dilinden esinlenmiştir. Ancak diğer programlama dillerinden farklı kılınmıştır, örneğin birçok programlama dilinde zorunluluk vardır, oysa Land'de zorunluluk yoktur.



İfadeler ve akış kontrolü

Land'den örnek ifadeler:



IF ifadesi, bir kod blokunu belli bir koşula bağlı olarak, else ve elif (else-if'ın kisaltmasi) ile birlikte çalıştırır.
FOR ifadesi, yinelenebilir bir nesne üzerinden yineleme yapar, bu sırada ilgili bloktaki her ögeyi bir yerel değişkene atar.
WHILE ifadesi, koşulu doğru olduğu sürece o kod blokunu çalıştırır.
FUNC ifadesi ile fonksiyon tanımlanır.
PRINT ifadesi bir ekrana yazdırma görevi görür.
